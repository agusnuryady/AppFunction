## How To RUn

1. Clone this repository
   ```
   $ git clone https://github.com/agusnuryady/AppFunction.git
   ```
2. Install all depedencies on the package.json
   ```
      cd ToDoApp
      npm install
   ```
3. Run Interviewers
   ```
   $ react-native run-android
   ```
